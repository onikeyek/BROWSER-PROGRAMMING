Lecture 6 – Asynchronous JavaScript

Timeout • Promise • Async/Await • Fetch

1. Async Timeout

Demonstrates that JavaScript does not wait. setTimeout schedules a task for later.

2. Async Promise

A Promise represents a value that will arrive in the future. .then() runs after it resolves.

3. Async / Await

Modern way to work with Promises. await pauses the function without freezing the browser.

4. Async Fetch

Uses fetch to get real data from an API. Includes error handling with try/catch/finally.
