# Naimot Yekini – Portfolio (Lecture 05)

Personal portfolio site built with HTML, CSS, and vanilla JavaScript.

## Features

- **Skills section** – 8 skill cards with icons and hover effects
- **Projects section** – 3 project cards with descriptions and GitHub links
- **Dark / Light theme toggle** – persisted via `localStorage` (`portfolio_theme`)
- **Click counter** – tracks button presses with a live label
- **Last updated** – auto-generated date injected by JavaScript on page load

## Files

| File | Purpose |
|------|---------|
| `index.html` | Structure and content |
| `style.css` | Styling, CSS variables, dark mode, responsive layout |
| `script.js` | Theme toggle, localStorage, counter, last-updated date |
